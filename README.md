# Eco-Cognition Shield

A browser and mobile tool that links Digital Addiction to Environmental Sustainability.

### 1. Web Application
The web application is built with React and Vite.

**Prerequisites:** Node.js installed.

1.  Open a terminal in the project folder.
2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser and go to: `http://localhost:5173`

### 2. Browser Extension
The Chrome extension tracks your actual usage on sites like YouTube and TikTok.

1.  Open Chrome and go to `chrome://extensions`.
2.  Enable **Developer mode** (toggle in the top right corner).
3.  Click **Load unpacked**.
4.  Select the `extension` folder inside this project directory:
   
5.  Visit YouTube or TikTok to see the overlay in action!
